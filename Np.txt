Нужна crm система для компании которая ханимается орагнизацией мероприятий. Внимательно изучи крупные аналоги YouGile в интернете. Примени полезныый функционал в найшей crm системе.  Изучи в интернете боли компаний которые занимаются органзацией мероприятий. 
Должен быть dashboard на котором будут видна основная аналитика с графиками по работе сотрудников.
Должны быть сделки, в сделках воронка продаж. с возможностью перемещать сделки курсором по воронке, создай карточку сделки. Нужны парамтетры: контакты, в каком зале будет проходить мероприятеи, время, дата, имя проводящего мероприятие, цена аренды, дополнительные комментарии, дополнительные задачи, сумма задатка, количество участников, автоформирование счёта. Нужен календарь со свободными слотами по залам, где можно отмечать день и временной промежуток аренды. Запись считается подтверждённой после внесения задатка.
Нужен раздел контакты. База клиентов (аредаторов) с учётом кто сколько раз проводил и на какую сумму. 
Нужен раздел по задачам. Нужна возможность по задачам ставить автонапоминание чтобы она могла приходить в телеграм.
Нужен раздел настройки, где можно настраивать воронки, стоимость аренды зала. Нужна возможность настраивать доступность залов и менять стоимость аренды.  Нужна возможность добавлять менеджеров по продажам, а также нужна возможность настройки интеграций с телеграм ботом, с telethon, для сбора заявок из заданного чата. Авторизация в telethon через qr код, добавь настройку с возможностью парсинга по ключевым словам и контекста с помощью AI. По подходящим сообщениям должна создаваться сделка в первом этапе воронки. 
Стек. Postgres, Reddis для кэша, pyton как основной язык.
Подготовь Docker контейнер с продакшн env файлом.
